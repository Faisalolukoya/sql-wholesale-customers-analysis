# Wholesale Customer Analysis Using SQL

## Project Overview
This project explores wholesale customer data using SQL to uncover purchasing patterns, customer spending behavior, and business insights.

## Objectives
- Analyze customer spending across product categories
- Identify top-spending customers
- Explore purchasing trends and patterns
- Generate insights to support business decision-making

## Tools Used
- SQL

## Dataset
Wholesale Customer Dataset

## Key Insights
Insights generated from SQL queries are documented in this repository.
