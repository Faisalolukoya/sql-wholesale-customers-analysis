# Wholesale Customer Analysis Using SQL

## Project Overview
This project analyzes wholesale customer data using SQL to uncover spending patterns, customer behavior, and regional trends. The analysis focuses on identifying high-value customers, comparing spending across regions, and exploring purchasing habits across product categories.

## Business Questions
1. Who are the top 5 customers with the highest total spending?
2. What is the average spending in each product category by region?
3. Which region has the highest average total spending?
4. Which customers spend above the average amount on Fresh products?
5. How many customers belong to each sales channel?
6. Which customers spend more on Grocery products than on Fresh and Milk combined?
7. How do customers rank within each region based on total spending?

## SQL Concepts Used
- Aggregate Functions (AVG, COUNT)
- GROUP BY
- ORDER BY
- Subqueries
- Window Functions (RANK)
- Filtering with WHERE
- Calculated Columns

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## Dataset
Wholesale Customers Dataset

## Key Findings
- Identified the top-spending customers across all product categories.
- Compared regional spending patterns using average category expenditure.
- Determined the region with the highest average total spending.
- Analyzed customer distribution across sales channels.
- Ranked customers within each region based on total expenditure.
